API Documentaion



This is the API documentation for the facts table. It contains the schema and route end points for the facts table.

facts end points is located at the following URL: http://your-server-name/facts

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /facts/all # list all facts
  • GET /facts/all/{fact_id} # get facts detail
  • GET /facts/filter?column={column_name}&value={value} # filter facts by single column
  • GET /facts/report?{column_name}={value} # report facts by single/multiple column using & operator
  • POST /facts/create # create one facts
  • POST /facts/create/list # create multiple facts
  • PUT /facts/update/{fact_id} # update facts by fact_id
  • PUT /facts/update?column={column_name}&value={value} # update facts by any column
  • DELETE /facts/delete/{fact_id} # delete one facts
  • DELETE /facts/delete # delete multiple facts
