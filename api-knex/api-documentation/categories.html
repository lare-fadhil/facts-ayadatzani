API Documentaion



This is the API documentation for the categories table. It contains the schema and route end points for the categories table.

categories end points is located at the following URL: http://your-server-name/categories

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /categories/all # list all categories
  • GET /categories/all/{category_id} # get categories detail
  • GET /categories/filter?column={column_name}&value={value} # filter categories by single column
  • GET /categories/report?{column_name}={value} # report categories by single/multiple column using & operator
  • POST /categories/create # create one categories
  • POST /categories/create/list # create multiple categories
  • PUT /categories/update/{category_id} # update categories by category_id
  • PUT /categories/update?column={column_name}&value={value} # update categories by any column
  • DELETE /categories/delete/{category_id} # delete one categories
  • DELETE /categories/delete # delete multiple categories
